# S.E.R.E.N.A. Bot
Custom Discord bot for the Tomb Raider Modding Server.

## Features:
 - Leveling system for:
   - Level
   - Cred
   - Assistance
 - Reputation command
 - Daily reward
 - Mod requests system
 - Dice command
 - CyberCookie and SpaceBurrito commands
 - Level up notification
 - Join message
