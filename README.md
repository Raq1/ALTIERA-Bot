# A.L.T.I.E.R.A. Bot
Custom Discord bot for the Cyberpunk 2077 Modding Server

## Features:
 - Leveling system for:
   - Level
   - Cred
   - Assistance
 - Reputation command
 - Daily reward
 - Dice command
 - CyberCookie and SpaceBurrito commands
 - Level up notification
 - Join message
